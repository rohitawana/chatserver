if you are not able to run this chat server by batch file you can run it manually by simple three steps as given below:


step 1: open cmd prompt type MyServer.java press enter
Step 2: open another cmd prompt type ClientLoggedOn.java press enter
Step 3: open one more cmd prompt type MyClient.java press enter

and enjoy chating u can have n number of client by running n number of MyClient.java on n number of cmd prompt


NOTE: u should have insyalled JDK in and JRE on your machine
